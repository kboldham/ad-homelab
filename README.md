# AD Home Lab — IT Support Portfolio

## About This Lab
Hands-on home lab documentation built on Proxmox VE covering core enterprise IT support skills.

## Labs
| Area | Topics |
|---|---|
| [Active Directory](./active-directory/) | AD DS, OUs, Users, Groups, GPOs |
| [Networking](./networking/) | DHCP, DNS, TCP/IP, OSI Troubleshooting |
| [PowerShell](./powershell/) | Cmdlets, Automation, AD Management |
| [Ticketing](./ticketing/) | osTicket, Ticket Prioritization |
| [Remote Support](./remote-support/) | RDP, Remote Administration |
| [Helpdesk Scenarios](./helpdesk-scenarios/) | Break/Fix, Troubleshooting Methodology |

## Environment
- Hypervisor: Proxmox VE
- Domain Controller: Windows Server 2022
- Client Machine: Windows 11
